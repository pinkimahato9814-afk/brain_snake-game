 <mark>BRAIN SNAKE GAME<mark> 

 # **Abstract**

 This project presents an innovative and educational Snake Game integrated with a quiz component, developed using Python and the Turtle graphics module along with the integration of databases. The classic Snake Game is combined with a series of quiz questions to enhance both entertainment and learning. This game focuses on a wide range of users including children to adults. The quiz covers a wide range of topics including science and technology, math, general knowledge ensuring fun learning along with enhancing game tactics.




# <mark>Introduction<mark>

1.1 Background:
The Brainy Snake game took its inspiration from the Classic Snake game introduced by Nokia in the late 1990s, the game featured a snake that moved around a small, pixelated screen, consuming food to grow longer while avoiding collisions with its own tail and the screen's edges. We improvised it by adding some more features to grab users attention and interactiveness. To be specific, by adding a quiz component to the game and enabling users to customize their game environment we tried to give the Classic Snake game a new perspective. The Brainy Snake game is more flexible in terms of dual role system : Player and Admin. Admin can customize the quiz questions making it suitable for all types of players.Data archiving is ensured by keeping records of user/admin, scores and questions.
